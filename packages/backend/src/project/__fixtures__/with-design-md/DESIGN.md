# Existing Project DS

> Category: Web Application
> Surface: web

A canonical design system that accompanies an existing project. Most values align
with what the code declares, but --color-accent intentionally differs so the
reconciliation path can be tested.

## 1. Visual Theme & Atmosphere

Dark surfaces with a blue accent. The palette is deliberately restrained.

## 2. Color

--color-surface: #0a0a0a
--color-accent: #aabbcc
--color-accent-hover: #8899bb
--color-border: #1a1a1a
--color-text: #fafafa

## 3. Typography

## 4. Spacing
