# Pick Your Foundation — Browsing, Previewing, Customizing

## User story

> As a **product designer**,
> I want to **browse a gallery of design system bases, see what each looks like,
> tweak the colors and fonts to match my brand, and create my own system**,
> So that **every component I build later automatically uses my brand identity.**

## Acceptance criteria

- The Catalog view shows 13 base design systems in a visual card grid
- Each card has a preview area, name, and category pill
- Category filter pills let me narrow by Editorial, Brutalist, Fintech, Deck, etc.
- A search bar filters bases by name, description, or category
- Clicking a card expands it to show token palette + rendered iframe preview
- A 5-step customization flow lets me set: identity, colors, typography, shape, review
- The preview iframe updates in real time as I customize
- Creating a design system writes tokens.css with my custom values
- I can switch between design systems at any time — all components re-skin

## Role

Product designer

## Effort

~3 minutes from browsing to a fully customized system
